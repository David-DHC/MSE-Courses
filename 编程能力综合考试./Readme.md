## Tips
- 数据量极小（肯定不超过1e6，甚至好像也不超过1e5），所以可以暴力求解。
- 允许使用STL（可以直接`#include <bits/stdc++.h>`）。
- 允许定义全局变量。
- 未特殊说明，整数均在`int`范围内。
- 输入输出中，同一行内不同数字均按空格分开，长度也会给出。不会出现必须使用`getline`或者`getchar`的地方。
